# Student Class

This file demonstrates how to build a simple **Student class** in Dart with constructors and methods.

##  Features
- **Default constructor** → creates a student with name, ID, and GPA.
- **Named constructor (`freshman`)** → creates a student with GPA defaulted to 0.0.
- **Fields**:
  - `name` → student’s name
  - `studentId` → unique ID
  - `gpa` → grade point average
- **Method**:
  - `printStudentInfo()` → prints student details.

## Learning Goals
- Practice constructors (default and named).
- Understand mutable vs immutable fields.
- Explore simple class design in Dart.
